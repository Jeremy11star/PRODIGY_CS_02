# PRODIGY_CS_02
Task 02: Pixel Manipulation for Image Encryption.

This is a Python project for the Prodigy InfoTech Cyber Security Internship.

The program encrypts and decrypts images. It manipulates the individual color values (pixels) of an image.

Key features:

The program loads any standard image format.

It converts the image to RGB mode.

It uses a simple mathematical shift key to scramble Red, Green, and Blue values.

The modulo operator ensures color values wrap correctly (0-255).

A negative key reverses the shift for complete decryption.

The tool saves the encrypted and decrypted images as separate files.

Technology used:

Python

Pillow (PIL) library
